功能与特性

  • 支持订阅 50 多种编程语言;
  • 支持添加/删除编程语言，并支持自定义它们的排序;
  • 支持收藏喜欢的项目;
  • 支持多种颜色主题自由切换;
  • 支持搜索,并自持自定义订阅关键字;
  • 支持分享,轻松将自己喜欢的项目分享给好友;

运行调试

  1. 准备React Native环境,可参考: Requirements。
  2. Clone GitHubPopular,然后终端进入项目根目录。
  3. 终端运行 npm i。
  4. 然后运行 react-native run-ios 或 react-native run-android。
  5. Ok,有问题可以提issues出来
