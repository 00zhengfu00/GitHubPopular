# GitHub Popular

这是一个用来查看GitHub最受欢迎与最热项目的App,它基于React Native支持Android和iOS双平台。

## 目录

* [功能与特性](#功能与特性)
* [下载安装](#下载安装)
* [预览图](#预览图)
* [ToDo](#todo)

## 功能与特性

* 支持订阅 50 多种编程语言;
* 支持添加/删除编程语言，并支持自定义它们的排序;
* 支持收藏喜欢的项目;
* 支持多种颜色主题自由切换;
* 支持搜索,并自持自定义订阅关键字;
* 支持分享,轻松将自己喜欢的项目分享给好友;
